This web application is a data science project investigating the effects of the COVID-19 pandemic in England. This project has been developed using a Python Flask backend, Heroku web hosting, and a CockroachDB cloud database. The source code can be found here. The data source can be found here.

The Summary page briefly shows daily case and death patterns over the course of the pandemic.

The Vaccines page assesses the effectiveness of vaccination on public health in England.

The Forecasting page uses time series analysis to predict how the pandemic will develop.

The Development page shows a list of features that I am planning to implement in the near future.
